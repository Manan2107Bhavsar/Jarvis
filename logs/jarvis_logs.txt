[20250824-193046] Jarvis: The problem asks to replace specific special characters with their HTML/XML entity equivalents without using built-in string replacement functions like `replace()` or `replaceAll()`.

This means we need to iterate through the string character by character, check if the character is one of the special ones, and if so, append its entity, otherwise append the character itself to a new result string (or a list of characters which we then join).

Here's how we can implement this:

1.  **Define a mapping:** Use a dictionary (hash map) to store the special characters and their corresponding entity replacements.
2.  **Iterate character by character:** Loop through each character in the input string.
3.  **Check for replacement:** For each character, check if it exists as a key in our mapping dictionary.
4.  **Build the result:**
    *   If the character is in the mapping, append its corresponding entity value from the dictionary to a list of parts.
    *   If it's not in the mapping, append the character itself to the list of parts.
5.  **Join the parts:** After iterating through all characters, join the list of parts into a single string. This is generally more efficient in Python than repeatedly concatenating strings.

```python
def escape_special_chars(s: str) -> str:
    """
    Replaces special characters in a string with their HTML/XML entity equivalents.
    Does not use built-in string replacement functions like replace() or replaceAll().

    Args:
        s: The input string.

    Returns:
        The string with special characters escaped.
    """
    # Define the mapping of special characters to their HTML/XML entities
    replacements = {
        '&': '&amp;',
        '<': '&lt;',
        '>': '&gt;',
        '"': '&quot;',
        "'": '&apos;' # Often &#39; for HTML, but &apos; for XML/XHTML
    }

    # Use a list to build the new string parts efficiently
    # (appending to a list and then joining is faster than repeated string concatenation)
    result_parts = []

    for char in s:
        # Check if the current character is one of the special ones
        if char in replacements:
            # If it is, append its corresponding entity
            result_parts.append(replacements[char])
        else:
            # If not, append the character as is
            result_parts.append(char)

    # Join all the parts to form the final escaped string
    return "".join(result_parts)
```
